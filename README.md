#  Gesture Slice — ML Hand-Powered Fruit Ninja

A real-time ML game controlled entirely by hand gestures.
Powered by **Google MediaPipe** — runs 100% in your browser. No install. No server. No cost.

🎮 **[Play it Live] https://shimra123.github.io/gesture-slice/ ⭐ 


---

## ⚠️ Before You Play

- Use **Chrome** or **Edge** only (not Firefox)
- Click **Allow** when camera permission appears
- Good lighting + hand **30–60cm** from camera
- No popup? → Click 🔒 in address bar → Camera → Allow → Refresh

---

## ✋ How to Play

| Gesture | Action |
|---------|--------|
|     Swipe index finger fast | Slice fruits |
| 👊 Make a fist | Pause / Resume |
| 💣 Avoid bombs | Lose a life or 5 seconds |

---

## 🎮 Game Modes

- 🍉 **Classic** — survive as long as possible
- ⏱️ **Time Attack** — max score in 60 seconds
